**Note:** Since it is a markdown file, the content is written in markdown format.
The content is converted to HTML format using the markdown parser.

### 1. Create a new post

- To create a new post, create a new markdown file in the `src/assets/posts` directory.
- Add the front matter to the markdown file if you want to add metadata to the post.
- Write the content of the post in markdown format.

### 2. MARKDOWN CHEATSHEET

**Headings:**

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6
```
---

```markdown

```

